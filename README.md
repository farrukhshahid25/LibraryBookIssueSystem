# Library Book Issue System 📚

A Java desktop application built to practice Object-Oriented Programming (OOP) and Exception Handling. 

## 👨‍💻 About The Project
This project simulates a basic library book issue interface. The main focus of this project was to implement strict form validation and handle various user inputs safely using Java's Exception Handling mechanisms.

## 🚀 Learning Outcomes & Features
* **GUI Development:** Built a responsive interface using Java Swing (`JFrame`) with custom styling.
* **Custom Exceptions:** Created specific error classes (e.g., `EmptyFieldException`, `InvalidDateException`) to handle custom logic errors.
* **Data Validation:** Used multiple `try-catch` blocks to prevent the application from crashing on invalid inputs (like entering alphabets in a numeric field).
* **User Feedback:** Integrated `JOptionPane` to show clean error and success pop-ups.

## 🛠️ Tech Stack
* Java (JDK)
* Java Swing & AWT

## ⚙️ How to Run
1. Ensure Java is installed on your system.
2. Clone this repository:
   ```bash
   git clone [https://github.com/farrukhshahid25/LibraryBookIssueSystem.git](https://github.com/farrukhshahid25/LibraryBookIssueSystem.git)
   ```
3. Compile the Java file:
   ```bash
   javac LibraryBookIssueSystem.java
   ```
4. Run the application:
   ```bash
   java LibraryBookIssueSystem
   ```
